ID3介紹¶

資訊滳(Entropy)¶

  • 衡量一個空間中資料分布的均勻程度。資料分布的越均勻，滳就越大；反之滳就越小。
  • 衡量單一資料的不確定性機率可用$-log (p_i)$
  • 但由於資料可能會有多種情況，因此在計算整體資料的不確定性時。應該計算整體不確定性$-log(p_i)$的期望值$E$，也就是資訊滳。其公式如下 $$E[-log p_i] = \sum\limits_{i=1}^n(p_i\ log\ p_i)$$
  • 資訊滳是對於事物不確性的衡量標準

資訊增益(Infomation Gain)¶

  • 對於每一項特徵項目計算平均資訊期望值，假定某特徵項目I有兩個特徵值(p1、p2)，其平均資訊期望值計算如下
    • 先計算p1在整體資料S中佔的比率$p1\over S$
    • 之後計算在p1的情況下S1類的次數C1及p1的情況下S2類的次數C2
    • 有了p1情況下各類別次數之後計算p1情況的資訊滳，即$p1\_I(S1,S2) = I(C1,C2) = -{{C1}\over p1}log{{C1}\over p1}-{{C2}\over p1}log{{C2}\over p1}$
    • 以同樣的步驟計算特徵值p2的資訊滳$p2\_I(S1,S2)$
  • 對於該特徵類別有了兩個特徵值的類別資訊滳之後即可計算平均資訊期望值，即 $$E(特徵項目) = {p1\over S} \times p1\_I(S1,S2)+{p1\over S}\times p1\_I(S1,S2)$$
  • 資訊增益 $$Information\ Gain(特徵項目) = S_I(S1,S2) - E(特徵項目)$$

決策迴歸樹¶

  • 傳統的決策樹是一種classification的模型
  • 經改良過後的決策樹也能進行數值型資料的預測
  • 決策迴歸樹
    • 選擇一個對目標欄位較有影響的特徵進行切割
    • 根據所選的特徵分割數據
    • 繼續選擇特徵進行切割
    • 預測.每個葉節點都會給出一個預測值. 這個值是所有訓練樣本的平均值

## 範例

from sklearn.tree import DecisionTreeRegressor 
from sklearn.model_selection import train_test_split
import numpy as np

X = np.arange(24, 36)   #氣溫
Y = np.array([22, 25, 28, 32, 32, 35, 34, 36, 34, 38, 42, 45])  #冰品銷售量

X = X.reshape(-1, 1)  #轉換為sklearn模型指定維度

X_train, X_test, y_train, y_test = train_test_split(X, Y, test_size=0.3)

tree = DecisionTreeRegressor()
tree.fit(X_train, y_train)

y_pred = tree.predict(X_test)
print('X test', X_test)
print('y pred', y_pred)

plt.figure(figsize=(20,20))
plot_tree(tree, filled=True, feature_names=['氣溫'])
plt.show()
